Smart House – Intelligent IoT System
STM32 · ESP32 · Multi-sensor · FastAPI

Design of a complete smart-home platform using STM32/ESP32 boards and multiple sensors (temperature, motion, light). Implementation of UART/Wi-Fi communication, data acquisition and forwarding to a FastAPI backend, plus a web interface for real-time monitoring and control of the environment.

STM32 ESP32 UART / Wi-Fi FastAPI Real-time Dashboard
Automatic Migration of Microchip Projects to STM32
AST Parsing · LLM · STM32 HAL

Development of a smart pipeline that parses legacy embedded code bases (Microchip, ESP, Arduino, PIC) using AST analysis and leverages large language models to automatically detect drivers, extract hardware features into JSON files, and generate mapping specifications toward STM32 HAL APIs. The goal is to accelerate firmware migration across MCU families.

Python AST Parsing LLM Integration JSON Mapping STM32 HAL
Biomedical ECG Sensor – ESP32
AD8232 · ADC · Filtering · Wireless

Integration of the AD8232 ECG front-end with an ESP32 microcontroller for cardiac signal acquisition. Implementation of ADC sampling, digital filtering for noise reduction and baseline stabilization, and UART/Wi-Fi transmission for real-time ECG waveform visualization and heart-rate analysis.

ESP32 AD8232 Signal Processing Wi-Fi / UART Biomedical
Industrial Cable Inspection – Vision AI
Jetson Nano · CNN · Anomaly Detection

Implementation of a convolutional neural network on a Jetson Nano platform to recognize industrial cables and detect visual anomalies. Automatic tuning of visual parameters such as saturation and contrast to maintain robust detection under varying lighting conditions.
